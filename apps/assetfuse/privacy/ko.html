개인정보 처리방침

AssetFuse

최종 수정일: 2026년 3월 9일

Jimin Kim(이하 "회사", "우리", "저희")은 AssetFuse(이하 "앱")을 상용 애플리케이션으로 개발하였습니다. 본 개인정보 처리방침은 귀하가 앱을 사용할 때 회사가 귀하의 정보를 수집, 사용 및 보호하는 방법에 대해 설명합니다.

AssetFuse을 사용함으로써 귀하는 본 방침에 따른 정보 수집 및 사용에 동의하게 됩니다.

1. 수집하는 정보

1.1 증권사 및 거래소 인증 정보

포트폴리오 동기화를 위해 앱은 지원되는 증권사 및 거래소의 API 인증 정보(API Key, API Secret)를 필요로 합니다:

  • 한국투자증권 — App Key, App Secret, 계좌번호
  • 업비트(Upbit) — Access Key, Secret Key
  • 빗썸(Bithumb) — API Key, Secret Key
  • 코인원(Coinone) — Access Key, Secret Key

이러한 인증 정보는 암호화된 시스템 수준의 보안 저장소인 macOS 키체인에만 저장됩니다. 저희 서버로 전송되지 않습니다.

1.2 포트폴리오 및 거래 데이터

계정을 동기화하면 앱은 다음 데이터를 SwiftData를 사용하여 기기에 로컬로 저장합니다:

  • 보유 종목 (자산명, 수량, 매입가, 현재가)
  • 거래 내역 (매수/매도 일자, 수량, 가격, 수수료)
  • 계좌 요약 (총 평가액, 손익)

이 데이터는 귀하의 기기에만 저장되며 외부 서버로 업로드되지 않습니다.

1.3 AI 채팅 데이터 (Max 등급만 해당)

AI 채팅 기능을 사용하는 경우, 귀하의 메시지와 포트폴리오 맥락 정보가 귀하가 선택한 AI 제공업체로 전송됩니다:

  • OpenAI (GPT 모델) — OpenAI 개인정보 처리방침 적용
  • Anthropic (Claude 모델) — Anthropic 개인정보 처리방침 적용

귀하의 AI API 키는 macOS 키체인에 저장됩니다. 저희는 귀하의 채팅 대화 내용이나 AI API 키를 저장, 기록하거나 접근하지 않습니다.

1.4 구독 정보

저희는 인앱 구매를 위해 Apple의 StoreKit을 사용합니다. 모든 결제 처리는 Apple이 담당합니다. 저희는 결제 정보(신용카드 번호, 청구 주소 등)를 수집하거나 저장하지 않습니다. 구독 상태를 확인하기 위한 거래 영수증만 수신합니다.

1.5 수집하지 않는 정보

  • 개인 식별 정보 (이름, 이메일, 전화번호)
  • 위치 데이터
  • 기기 식별자 또는 광고 ID
  • 사용 분석 또는 행동 추적
  • 연락처, 사진 또는 기타 기기 데이터

AssetFuse은 계정 생성이나 로그인을 요구하지 않습니다.

2. 정보의 사용 방법

데이터 목적 저장 위치
증권사 API 인증 정보 포트폴리오 동기화를 위한 증권사/거래소 API 인증 macOS 키체인 (기기 내, 암호화)
포트폴리오 및 거래 데이터 보유 종목 표시, 손익 계산, 보고서 생성 SwiftData (기기 내)
AI 채팅 메시지 분석을 위해 선택한 AI 제공업체로 전송 OpenAI 또는 Anthropic으로 직접 전송; 저희가 저장하지 않음
구독 영수증 구독 등급 확인 (Free / Pro / Max) Apple StoreKit에서 관리

3. 제3자 서비스

앱은 귀하를 대신하여 귀하의 API 인증 정보를 사용하여 다음 제3자 서비스와 통신합니다:

서비스 목적 전송 데이터
한국투자증권 API 포트폴리오 동기화, 거래 내역 조회 귀하의 API 인증 정보 (HTTPS 통해)
업비트 API 암호화폐 포트폴리오 동기화, 거래 내역 조회 귀하의 API 인증 정보 (HTTPS 통해)
빗썸 API 암호화폐 포트폴리오 동기화, 거래 내역 조회 귀하의 API 인증 정보 (HTTPS 통해)
코인원 API 암호화폐 포트폴리오 동기화, 거래 내역 조회 귀하의 API 인증 정보 (HTTPS 통해)
OpenAI API (선택 사항) AI 기반 포트폴리오 분석 귀하의 API 키, 채팅 메시지, 포트폴리오 요약
Anthropic API (선택 사항) AI 기반 포트폴리오 분석 귀하의 API 키, 채팅 메시지, 포트폴리오 요약

모든 통신은 HTTPS 암호화를 사용합니다. 저희는 프록시 서버를 운영하지 않으며, 귀하의 기기가 이러한 서비스에 직접 연결됩니다.

4. 데이터 저장 및 보안

4.1 기기 내 저장

  • 민감한 인증 정보 → macOS 키체인 (AES-256 암호화, Apple Silicon에서 하드웨어 기반)
  • 포트폴리오 데이터 → SwiftData (로컬 SQLite 데이터베이스)
  • 클라우드 동기화 없음 — 모든 데이터는 귀하의 기기에 남아 있습니다

4.2 보안 조치

  • App Sandbox 활성화 (macOS 보안 경계)
  • Hardened Runtime 활성화
  • 모든 네트워크 통신 HTTPS/TLS 사용
  • 저희가 운영하는 원격 서버 없음
  • 외부로 전송되는 데이터 원격 측정 또는 충돌 보고 없음

4.3 데이터 침해

모든 데이터가 귀하의 기기에 로컬로 저장되고 저희가 사용자 데이터를 저장하는 서버를 운영하지 않으므로, 중앙 집중식 데이터 침해의 위험이 제거됩니다. 귀하의 데이터 보안은 macOS 기기 보안(비밀번호, FileVault 등)에 달려 있습니다.

5. 데이터 보관 및 삭제

  • 귀하의 데이터는 귀하의 기기에 남아 있습니다. 저희는 사본을 보유하지 않습니다.
  • 앱 내에서 언제든지 개별 증권 계좌와 관련 데이터를 삭제할 수 있습니다.
  • 앱을 제거하면 로컬에 저장된 모든 포트폴리오 데이터가 삭제됩니다.
  • 키체인 항목은 앱 제거 후에도 남아 있을 수 있으며, 키체인 접근을 통해 수동으로 제거할 수 있습니다.

6. 아동의 개인정보

AssetFuse은 17세 미만을 대상으로 하지 않습니다. 저희는 아동의 정보를 고의로 수집하지 않습니다. 본 앱은 금융 투자 데이터를 다루며 성인 사용자를 대상으로 합니다.

7. 내보내기 기능

앱을 통해 포트폴리오와 거래 내역을 CSV 또는 XLSX 파일로 내보낼 수 있습니다. 내보낸 파일은 귀하가 선택한 기기의 위치에 저장됩니다. 저희는 이러한 파일을 수신하거나 전송하지 않습니다.

8. 귀하의 권리

저희가 서버에 귀하의 개인 데이터를 수집하거나 저장하지 않으므로, 전통적인 정보 주체의 권리(접근, 수정, 삭제 요청)는 귀하가 기기에서 직접 관리합니다. 앱 내의 모든 데이터에 대해 완전한 통제권을 가집니다.

제3자 AI 서비스(OpenAI, Anthropic)를 사용하는 경우, 해당 서비스가 처리하는 데이터에 대한 귀하의 권리는 각 서비스의 개인정보 처리방침을 참조하시기 바랍니다.

9. 면책 조항

AssetFuse은 포트폴리오 추적 및 분석 도구입니다. 투자 조언, 재무 설계 또는 매매 추천을 제공하지 않습니다. 손익 계산 및 AI 생성 분석을 포함한 모든 표시 데이터는 정보 제공 목적으로만 사용됩니다. 투자 결정을 위해서는 공인 재무 상담사에게 문의하시기 바랍니다.

과거 실적이 미래 수익을 보장하지 않습니다. 암호화폐 및 증권 투자에는 손실 위험이 수반됩니다.

10. 본 개인정보 처리방침의 변경

저희는 수시로 본 개인정보 처리방침을 업데이트할 수 있습니다. 변경 사항은 본 문서 상단의 "최종 수정일"을 업데이트하여 반영됩니다. 변경 후 앱을 계속 사용하는 것은 개정된 방침에 대한 동의로 간주됩니다.
