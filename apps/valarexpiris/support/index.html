FAQ

How are my API keys stored?

API key values are stored in the macOS/iOS Keychain, Apple's secure credential storage. They are never saved as plain text in the app's database.

How does sync work?

Metadata syncs via iCloud (CloudKit) and key values sync via iCloud Keychain. Both are managed entirely by Apple's infrastructure.

Can I use the app without iCloud?

Yes. The app works fully offline. iCloud sync is optional and depends on your device settings.

How do expiration alerts work?

Set an expiration date when adding a key. The app schedules local notifications: a reminder before expiry (configurable 1–30 days), an alert on the day, and follow-ups for 3 days after.

I accidentally deleted a key. Can I recover it?

Deleted keys cannot be recovered. Use the Archive feature instead of deleting to safely retire keys while keeping them recoverable.

The app is not syncing across devices.

Make sure iCloud is enabled in System Settings and that the app has iCloud permission. Sync may take a few moments depending on network conditions.
